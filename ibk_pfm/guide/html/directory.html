0. 공통

  • 실서버와 동일한 구조로 구성
  • HTML : /jsp/phone/pfm/
  • CSS : /css/phone/pfm/
  • IMAGE : /img/phone/pfm/
  • JS : /js/lib/


1. HTML

  • HTML 문서는 화면정의서의 화면아이디와 동일하게 정의함
  • 메뉴별 폴더는 /pfm/아래 구분하여 파일 분류
  • ※ 2Depth 이하 HTML파일 업무코드 참고

​

2. CSS

  • [공통] common.css : reset 설정, 공통 속성 설정, 공통 기본 요소(버튼, 폼요소 등) 스타일 정의 ※ 모든 화면 호출
  • [공통] contents.css : 서브 화면 공통요소 스타일 정의 ※ 모든 화면 호출
  • [공통] swiper.css : swiper 플러그인의 기본 스타일
2-1. PFM 구축 CSS
  • com.css : 개인자산관리 공통 스타일 정의 ※ 모든 화면 호출
  • fin.css : My자산 스타일 정의
  • ies.css : My지출 스타일 정의
  • poa.css : My리포트/자산 모으기/연금 스타일 정의
  • cma.css : 신용관리 스타일 정의
  • pdr.css : 자산 모으기 > 잔돈저축 스타일 정의
  • res.css : 부동산 스타일 정의
  • gog.css : 정부지원금 스타일 정의
  • cam.css : 커리어관리 스타일 정의
  • dic.css : 오늘의 금융팁 스타일 정의
  • main.css : 개인자산관리 메인, 서브메인 스타일 정의
  • animation.css : 개인자산관리 모션 스타일 정의
​

3. IMAGE

3-1. PFM 구축 IMAGE
  • pfm : 개인자산관리 공통 요소에 사용되는 이미지
  • /pfm/아래 fin / ies / ... : 개인자산관리 서브 화면에서 사용되는 이미지 - 메뉴별 폴더구분하여 파일 분류
  • /pfm/man : 개인자산관리 메인 요소에 사용되는 이미지

  • [공통] jquery-lib.js : jQuery 라이브러리(jquery-1.8.3.min) ※ 모든 화면 호출

  • swiper.min.js : swiper 플러그인
